# Privacy Policy

Last updated: 2026-04-08

This policy describes how `fhir-format` handles data.

## Summary

`fhir-format` is a Claude plugin skill distributed as documentation and skill instructions.
It does not run a hosted backend service and does not independently collect, store, or sell personal data.

## What this plugin does

- Provides guidance for FHIR formatting and reasoning.
- Uses repository-hosted skill and documentation files.

## Data handling

- The plugin itself does not include telemetry, analytics collection, or user-tracking logic.
- Any prompt/response handling is performed by the platform where the plugin is used (for example, Claude), under that platform's policies.
- If users run local CLI tools referenced by this project, processing occurs in the user's local environment unless they separately configure external services.

## Protected health information (PHI)

- Do not include real PHI in public issues, benchmark artifacts, or example files.
- This project includes synthetic examples only.

## Third-party services

When you access repository hosting or documentation hosting (for example, GitHub and GitHub Pages), those providers may process technical logs per their own policies.

## Changes

This policy may be updated over time. Material changes will be reflected by updating the date above.
