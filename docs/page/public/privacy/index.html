Privacy Policy

Last updated: 2026-04-08

Summary: `fhir-format` is a Claude plugin skill distributed as repository content. It does not run an independent hosted backend and does not independently collect, store, or sell personal data.

What this plugin does

  • Uses repository-hosted skill and documentation files.

Data handling

  • Prompt and response handling is performed by the platform where the plugin is used.
  • Local CLI examples run in the user's environment unless external services are explicitly configured.
